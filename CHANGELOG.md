# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.1.0] - 2026-03-23

### Added

- **74 coding problems** across categories: algorithms, FastAPI, Django, Pytest fixtures/mocking, SQL, and problem-solving
- **Python sandbox** for secure, isolated code execution with resource limits
- **AI mock interview** powered by OpenRouter (Claude) — senior interviewer persona
- **Scored feedback** with detailed evaluation of code and interview performance
- **Show Answer** — AI-generated solutions streamed character-by-character via SSE
- **Code Chat** — in-editor AI chatroom for questions about problems or code
- **Ask AI** — select code in the editor and ask AI for explanations
- **Session management** — create, save, and review coding sessions with history
- **Monaco Editor** integration with Python syntax highlighting
- **Docker support** via `docker-compose` for one-command setup
- **Playwright E2E tests** for critical user flows
- **Vite dev server** with HMR and API proxy configuration
